## ‚ùå What Users HATE / Complain About

### 1. **Limited to 8 Slots** (MAJOR COMPLAINT)
- **Feedback:** Implied in multiple reviews asking "How many plugins?"
- **Problem:** Only 8 plugin slots per chain ‚Äî too limiting for complex processing
- **Impact:** Users hit ceiling quickly with mastering chains, complex vocal processing

**ProChain advantage:** ‚úÖ UNLIMITED plugins in our tree-based system

---

### 2. **Linear Chain Only (No Advanced Routing)**
- **Problem:** Can't do parallel processing, mid/side, or complex signal flow
- **Comparison:** "Waves StudioRack allows parallel chains" ‚Äî users mention this as advantage
- **Impact:** Can't do advanced techniques like parallel compression, frequency splitting

**ProChain advantage:** ‚úÖ Full serial/parallel groups with nested routing

---

### 3. **No Sidechain Routing**
- **Problem:** Can't route external sidechain inputs to compressors/gates
- **Impact:** Missing essential mixing technique (ducking, dynamic EQ triggers)

**ProChain status:** ‚ùå We need to add this (already in Phase 3 roadmap)

---

### 4. **Price vs Features** (Controversial)
- **Original price:** $49 USD
- **Common sale price:** $5-25 USD (often 80-90% off)
- **Feedback:** Many users say "wait for sale" or "only worth it on sale"
- **Comparison:** Blue Cat PatchWork ($99) does more but costs more

**ProChain strategy:** Position as premium tool ($79-99?) with far more features, or freemium with cloud sync premium tier

---

### 5. **DAW Already Does This** (Biggest Criticism)
- **Feedback:** "My DAW already has track groups/VCA faders/bus processing"
- **Examples:**
  - Logic Pro: Track Stacks + Summing Stacks
  - Ableton: Instrument/Audio Racks + Group Tracks
  - Studio One: Track Groups + FX Chains
- **Problem:** Power users don't see the value if their DAW has routing

**ProChain advantage:** ‚úÖ Our tree-based routing is MORE powerful than DAW routing, plus cloud sharing

---

### 6. **No Cross-DAW Compatibility** (Limitation)
- **Problem:** KSHMR Chain presets are plugin-specific, not DAW-specific
- **But:** Can't transfer entire DAW sessions with chains intact
- **User need:** "I want to move my vocal chain from Logic to Ableton"

**ProChain advantage:** ‚úÖ Cloud-based chain library works across DAWs
